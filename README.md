# XR EULA Prototype - Child-Friendly VR Agreement Experience

## Overview

This project is an interactive XR prototype developed in Unity for the Meta Quest platform. It explores how End User License Agreements (EULAs) can be redesigned to be more understandable and engaging for children.

The prototype transforms complex XR terms (such as safety, data privacy, and consent) into an interactive VR experience using simple language, visuals, audio, and guided navigation.

This work is part of my graduate research on children's interaction with XR technologies and digital agreements. 

---

## Concept

Instead of presenting long legal text, this prototype uses a **game-like structure**:

* A central hub with doors representing key topics
* Interactive rooms explaining each concept
* A mini-game to reinforce engagement and understanding

The goal is to help children:

* Understand safety rules
* Learn what data is collected
* Make more informed decisions before agreeing

---

## Key Features

**Door-based navigation system**
**Interactive information rooms**

  * Physical Safety (Do’s and Don’ts)
  * Data Privacy and Content Awareness
  * Legal and Agreement Understanding
* **Audio narration for accessibility**
* **Simple visuals and child-friendly language**
* **Mini-game (garden interaction) to reinforce learning**
* **Summary + screenshot feature for parent review**

---

## Tech Stack

* Unity (2022.x)
* C#
* XR Interaction Toolkit
* Meta Quest (VR)

---

## How to Run

**This project requires a Meta Quest headset**

1. Clone the repository:
   git clone https://github.com/upamaInBits/XR-Prototype.git

2. Open Unity Hub

3. Open this project using Unity Editor (2022.3.x recommended)

4. Connect your Meta Quest device

5. Build and run the project to the headset

## Research Context

This prototype was designed using:

* Co-design sessions with children
* Parent interviews
* Analysis of XR EULAs

It implements child-centered communication strategies such as:

* Simple language
* Visual explanations
* Chunked, interactive information delivery 

---

## Purpose

The goal of this project is to:

* Improve children's understanding of digital agreements
* Explore better UX for XR consent systems
* Reduce risks related to safety, privacy, and unintended decisions
